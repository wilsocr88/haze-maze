# Haze Maze

Mister Dizzy - Present Haze: interactive album

## Requirements

* [Java](https://www.java.com/)

## How to play

The ```dist/``` directory contains everything you need. Just run ```Haze.bat``` for Windows, or ```Haze.sh``` for Mac OS and Linux.
